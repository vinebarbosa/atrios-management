A sidebar navigation row — leading product/status dot + label. `active` applies the selected fill and heavier, brighter text. Rest state uses Inter's 450 "book" weight.

```jsx
<SidebarItem label="Pórtico" color="var(--atr-status-done)" active />
<SidebarItem label="Cortina" color="var(--atr-status-progress)" />
```
